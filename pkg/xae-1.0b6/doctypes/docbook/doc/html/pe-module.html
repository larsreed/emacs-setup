%*.module; Parameter Entities

Name

%*.module; Parameter Entities -- Control element definitions

Synopsis

The module parameter entities provide marked sections around element definitions. You can selectively include or remove elements from DocBook by changing these parameter entities.

Description

Each of the module parameter entities has the following expansion:

  INCLUDE

The following module parameter entities are defined in DocBook:
