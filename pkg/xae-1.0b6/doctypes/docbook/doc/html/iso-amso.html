Added Math Symbols: Ordinary Character Entities (%isoamso;)

Name

%isoamso; -- Added Math Symbols: Ordinary Character Entities

Synopsis

The %isoamso; parameter entity includes the ISO character entities with the public identifier: 

  ISO 8879:1986//ENTITIES Added Math Symbols: Ordinary//EN

The following character entities are defined in this entity set:
