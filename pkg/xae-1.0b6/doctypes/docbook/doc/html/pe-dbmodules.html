%db*; Parameter Entities

Name

%db*; Parameter Entities -- Control inclusion of DocBook modules

Synopsis

These parameter entities control the inclusion of DocBook modules.

Description

The modules listed here are included in DocBook DTD via parameter entity reference. For more information about these modules, and additional modules that can be defined, see Chapter 5.

%dbpool;

Points to the information pool module via its public identifier:

  -//Davenport//ELEMENTS DocBook Information Pool V3.0//EN
%dbhier;

Points to the hierarchy module via its public identifier:

  -//Davenport//ELEMENTS DocBook Document Hierarchy V3.0//EN
%dbgenent;

Points to the general entity module via its public identifier:

  -//Davenport//ENTITIES DocBook Additional General Entities V3.0//EN
