Added Math Symbols: Arrow Relations Character Entities (%isoamsa;)

Name

%isoamsa; -- Added Math Symbols: Arrow Relations Character Entities

Synopsis

The %isoamsa; parameter entity includes the ISO character entities with the public identifier: 

ISO 8879:1986//ENTITIES Added Math Symbols: Arrow Relations//EN
